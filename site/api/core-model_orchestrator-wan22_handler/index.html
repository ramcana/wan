    core.model_orchestrator.wan22_handler¶

    WAN2.2-specific model handling and processing.

    This module provides specialized functionality for WAN2.2 models including: - Sharded model support with selective downloading - Model-specific conditioning and preprocessing - Text embedding caching for multi-clip sequences - VAE optimization and memory management - Development/production variant switching - Input validation for different model types
