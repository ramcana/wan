    core.model_orchestrator.tests.test_cross_platform_compatibility¶

    Cross-platform compatibility tests for Model Orchestrator.

    Tests Windows, WSL, and Unix-specific behaviors including: - Path handling and long path support - File system operations and atomic moves - Lock mechanisms and process synchronization - Case sensitivity and reserved names
