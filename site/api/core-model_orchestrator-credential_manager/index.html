    core.model_orchestrator.credential_manager¶

    Secure credential management system for the Model Orchestrator.

    This module provides secure storage and retrieval of credentials using system keyring, environment variables, and secure configuration files. It also handles credential masking in logs and supports presigned URLs and temporary access tokens.
